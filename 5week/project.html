프로젝트 기획안

1. 프로젝트 이름/설명 건강한 습관(Healthy habits)

2. 페이지 구성


1) 메인페이지

- 증상별 아이콘

- 검색창

2) 본문1 (증상별)

- BEST10 상품 나열

* 노출되는 정보 : 상품 이미지, 상품 이름, 브랜드, 가격, 주문페이지 링크

** 데이터 출처 : 위케어 (https://shallwecare.com/)

3) 본문2 (검색창)

- 검색단어와 일치하는 상품 노출

- 일치하는 상품 없을 경우 유사한 단어 추천


3. 기획의도

- 코로나19 등으로 국민들의 건강에 대한 관심 급증

- 영양제 섭취를 통한 질병 사전예방 및 건강한 습관 만들기

4. 개발 해야 하는 기능들

- 증상 아이콘을 클릭하면, 관련 상품 BEST10을 나열한다.

- 증상, 브랜드명 등 검색어 직접 입력 시 유사한 상품을 노출한다.

5. 개발 계획

6주차 : 데이터값 크롤링하기

7주차 : 서비스 화면 디자인하기
